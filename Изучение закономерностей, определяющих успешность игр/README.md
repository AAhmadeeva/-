## Описание проекта

Используя исторические данные о пользователях и рекламных активностях, провести исследование поведения игроков в зависимости от источника перехода.


## Навыки и инструменты
- Python
- Pandas
- Matplotlib
- Предобработка данных
- Исследовательский анализ данных
- Проверка статистических гипотез
- Построение дашбордов
- Tableau

## Общий вывод

Были обработаны данные, проанализированы действия игроков и каналы привлечения, определены наиболее эффективные рекламные источники, проведены статистические тесты.
